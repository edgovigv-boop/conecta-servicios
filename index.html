Aviso de Privacidad

Conecta Servicios recopila nombre público, teléfono, ubicación general y descripción de la publicación para mostrar registros locales y facilitar contacto por WhatsApp. El teléfono se muestra parcialmente oculto dentro de la app.

Si el usuario autoriza la ubicación, se usa de forma aproximada para ordenar publicaciones cercanas. No se muestra la ubicación exacta de usuarios a otras personas y puede publicarse sin activar GPS.

Este aviso es una base para prueba pública. Antes de operación masiva conviene revisión legal formal.

Términos y Condiciones

La aplicación conecta personas de forma local. Cada usuario es responsable de verificar condiciones, pagos, horarios y seguridad antes de realizar cualquier acuerdo.
