# Conecta Servicios v4.8.2 — Filtros, chat y cursos

Actualización autorizada para Vercel/Netlify.

## Cambios incluidos
- Publicaciones cerca de mí ya no oculta registros: muestra todos los activos y ordena cercanos primero si hay ubicación.
- Cuando el filtro dice Todo México, el selector de municipios queda inhibido para no abrir miles de municipios.
- Buscador con un solo icono y campo desbloqueado.
- Home reforzado visualmente con el estilo aprobado.
- Mensajes inteligentes con chat piloto dentro de la app.
- Aprendizaje convertido a cursos gratuitos destacados: Fundación Carlos Slim, Google Actívate, Crece con Google e IBM SkillsBuild.
- Cache/PWA actualizado para forzar que el celular tome la nueva versión.

## Validación rápida
1. Abrir Publicaciones cerca de mí: debe mostrar el total de registros activos, no solo 3.
2. En Todo México: municipio debe quedar deshabilitado.
3. Elegir un estado: municipios se habilitan.
4. Buscar texto: debe escribir normal y filtrar.
5. Entrar a Mensajes: debe permitir escribir y responder dentro de la app.
6. Entrar a Aprendizaje: debe mostrar cursos gratuitos.
