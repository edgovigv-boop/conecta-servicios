# Conecta Servicios v4.6.3 — Rutas claras, home simple y mensaje dinámico

Versión consolidada con ajustes de estructura y experiencia:

- Home más intuitiva y con menos texto.
- “Oportunidades para ti” como acción principal.
- “Publicaciones cerca de mí” como acceso a resultados reales.
- Botón “+ PUBLICA” más visible.
- Micro-mensaje dinámico junto al contador de registros.
- Oportunidades para ti ahora funciona como rutas guiadas, no como filtros mezclados.
- “Quiero empezar algo propio” agrupa ideas de servicios, alimentos, limpieza, entregas, reparaciones y publicación inicial.
- “Tengo un negocio” guía a publicar negocio y buscar mensajeros locales.
- “Aprende y emprende” queda como sección independiente con cursos básicos y profesionales.
- Mantiene analítica, PWA, reclasificación admin, enlaces directos y cercanía inteligente.

No requiere SQL nuevo si ya se aplicaron las versiones anteriores.
